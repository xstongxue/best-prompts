# 周报撰写 Prompt

# 使用说明

将下方 Prompt 完整复制到聊天框，然后在 `[在此处粘贴草稿]` 位置填入你的工作草稿内容（流水账、碎片笔记、表格数据均可），AI 将整理为标准高质量周报。

若**没有草稿**，删除 `# 草稿` 章节，改为在 `# 背景信息` 中描述本周工作重点，AI 将根据描述组织内容。

---

# 写作质量标准

## 内容层面（必须满足）

1. **有结论，不只有陈述**
   - 🚫 差：`完成了模型测试`
   - ✅ 好：`完成 Qwen3.5-122B-A10B-AWQ 在 4×24G NVIDIA 环境下的高考模拟卷测试，错题率 1/18，推荐作为 NVIDIA 多卡方案首选`

2. **数据具体**：保留时间、数量、百分比等一切可量化的信息，不用"大幅""显著"等模糊词代替

3. **问题定位三要素**：现象 → 根因 → 影响范围，缺一不可

4. **计划可执行**：下周计划写明具体目标和时间节点，不写"继续推进""加强沟通"类空话

## 结构层面

- 同类工作项优先用**表格**对比展示（如多模型测试结果）
- 重要判断或推荐用**引用块（`>`）**突出
- 不同业务线用**小标题**区分，避免堆砌成一个大列表

## 语言层面

- 主动语态，简洁直接
- 专业术语、模型名称、参数名、配置规格原样保留
- 同一事物前后用词保持一致

# 草稿保留原则（有草稿时）

- 草稿中的所有表格、图片引用 `![描述](路径)`、数据、引用块**原样保留，不删减，不概括**
- 图片路径绝对不能修改
- 不虚构草稿中不存在的事实
